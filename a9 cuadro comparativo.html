CUADRO COMPARATIVO : PAGINA WEB ESTATICA Y DINAMICA

CARACTERISTICAS PÁGINA WEB ESTATICA PAGINA WEB DINAMICA
Definicion Una página web estática es un sitio compuesto por archivos HTML, CSS y JavaScript fijos, que se entregan al navegador del usuario exactamente igual que están almacenados en el servidor Una página web dinámica es un sitio interactivo cuyo contenido se genera en tiempo real, cambiando según el usuario, la hora, la ubicación o sus acciones
Ejemplos portafolios personales, páginas corporativas básicas, documentación técnica (ej. sitios generados con Hugo), landing pages, currículums en línea, blogs simples y sitios tipo "under construction" o galerías de imágenes, caracterizados por su alta velocidad y seguridad redes sociales (Facebook), sitios de comercio electrónico (Amazon), plataformas de streaming (Netflix) y gestores de contenido (WordPress)
Ventajas Las páginas web estáticas ofrecen ventajas principales como una velocidad de carga superior, mayor seguridad al no usar bases de datos y costes de alojamiento más económicos. Las páginas web dinámicas ofrecen ventajas clave como actualizaciones en tiempo real, alta interactividad, personalización de la experiencia del usuario y gestión sencilla de contenidos mediante bases de datos
Desventajas dificultad para actualizar contenido, ya que requiere edición manual de código, y la falta de interactividad real, al no utilizar bases de datos. mayores costos de desarrollo y mantenimiento, riesgos de seguridad elevados (vulnerabilidades en bases de datos), tiempos de carga más lentos debido al procesamiento del servidor y una mayor complejidad técnica en comparación con los sitios estáticos.
